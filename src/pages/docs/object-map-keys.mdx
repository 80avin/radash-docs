## Basic usage

Given an object and a mapping function, returns a new object with all the keys mapped.

```ts
import { mapKeys } from 'radash'

const ra = {
  mode: 'god',
  power: 'sun'
}

mapKeys(ra, k => k.toUpperCase()) // => { MODE, POWER }
```
