## Basic usage

Given an array of objects and an identity callback function to determine
how each object should be identified returns an object where they key
are the id values the callback returned and the values are an integer
telling how many times that id occurred.

```ts
import { counting } from 'radash'

const gods = [{
  name: 'Ra',
  culture: 'egypt'
}, {
  name: 'Zues',
  culture: 'greek'
}, {
  name: 'Loki',
  culture: 'greek'
}]

counting(gods, g => g.culture) // => { egypt: 1, greek: 2 }
```
