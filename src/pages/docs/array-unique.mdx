## Basic usage

Given an array of items -- and optionally, a function to determine their identity -- return a new array without any duplicates.

```ts
import { unique } from 'radash'

const fish = [{
  name: 'Marlin',
  weight: 105,
  source: 'ocean'
}, {
  name: 'Salmon',
  weight: 22,
  source: 'river'
}, {
  name: 'Salmon',
  weight: 22,
  source: 'river'
}]

unique(fish, f => f.name) // => [marlin, salmon]
```
