## Basic usage

Given an object returns a new object with the keys and values reversed.

```ts
import { invert } from 'radash'

const powersByGod = {
  ra: 'sun',
  loki: 'tricks',
  zeus: 'lighning'
}

invert(gods) // => { sun: ra, tricks: loki, zeus: lightning }
```
