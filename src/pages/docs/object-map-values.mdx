## Basic usage

Given an object and a mapping function, returns a new object with all the values mapped.

```ts
import { mapValues } from 'radash'

const ra = {
  mode: 'god',
  power: 'sun'
}

mapValues(ra, k => k.toUpperCase()) // => { mode: 'GOD', power: 'SUN' }
```
