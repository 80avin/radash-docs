## Basic usage

Given an array of items, create a dictionary with keys and values mapped by given functions.

```ts
import { objectify } from 'radash'

const fish = [{
  name: 'Marlin',
  weight: 105
}, {
  name: 'Bass',
  weight: 8
}, {
  name: 'Trout',
  weight: 13
}]

objectify(fish, f => f.name, f => f.weight) // => { Marlin: 105, Bass: 8, Trout: 13 }
```
