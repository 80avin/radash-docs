## Basic usage

Given an array of items and desired cluster count, returns an array containing `n` arrays (where n is the cluster count) with the items split as evenly as possible between each array.

```ts
import { cluster } from 'radash'

const gods = [{
  name: 'Ra',
  power: 100
}, {
  name: 'Zues',
  power: 98
}, {
  name: 'Loki',
  power: 72
}, {
  name: 'Vishnu',
  power: 100
}]

cluster(gods, 2) // => [[ra, zues], [loki, vishnu]]
```
